# Hospital Management System SQL PROJECT
![HOSPITAL MANAGEMENT SYSTEM](https://github.com/AgasthiDoshi/Hospital_Management_Database_SQL_PROJECT/blob/main/logos.png)
## Overview
This project presents a comprehensive SQL-based Patient Management System designed to simulate real-world healthcare data operations. It demonstrates key database design principles and showcases complex SQL usage for managing and analyzing patient health records. The database handles crucial patient details like name, gender, allergies, height, weight, date of birth, and regional identifiers, reflecting a robust foundation for medical data systems.
